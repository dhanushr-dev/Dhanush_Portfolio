## 🚀 Portfolio Tech Stack & Features

This portfolio itself is a testament to modern frontend practices:
*   **Framework:** [TanStack Start](https://tanstack.com/router/v1/docs/start/overview) + React 19 + TypeScript.
*   **Styling:** Tailwind CSS v4 for super-fast styling and beautiful, responsive grids.
*   **Animations:** Framer Motion for premium-feeling transitions, float animations, and page interactions.
*   **Interactive Terminal:** An on-screen mock command line (`CodeTerminal`) mimicking developer tools.
*   **PDF Generation:** Client-side dynamic One-Pager PDF generator using `jspdf` to download resume sheets on the fly.
*   **Theme Toggle:** Dark and Light mode support using standard state hooks.

## ⚡ Quick Start

Follow these steps to run the portfolio on your local machine:

### Prerequisites
Make sure you have [Node.js](https://nodejs.org/) installed (v18+ recommended).

### 1. Clone the repository
```bash
git clone https://github.com/dhanushr-dev/Dhanush_Portfolio.git
cd Dhanush_Portfolio
```

### 2. Install dependencies
```bash
npm install
```

### 3. Start the development server
```bash
npm run dev
```
The application will be running at [http://localhost:8080](http://localhost:8080).

### 4. Build for production
To build a production bundle ready for hosting:
```bash
npm run build
```
